# Defend The Truth Ministries

**Mission:** To equip and train young Christian men and women with the principles to boldly share and faithfully defend the Truth of God's Word, raising up leaders who stand with conviction on the Scriptures in a secular world.

**Vision:** To see a generation of God-fearing men and women boldly proclaiming and defending God's Truth across the world.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YegorCreative/defend-the-truth-ministries.git
   cd defend-the-truth-ministries
   ```

2. **Open `index.html` in your browser:**
   ```bash
   open index.html
   ```
   Or simply drag `index.html` into your preferred browser.

3. **Development:**
   - Edit HTML in `index.html`
   - Add styles in `styles.css` and `responsive.css`
   - Add interactivity in `script.js`
   - Store assets in the `/assets` directory

## Features

- **Responsive Design:** Mobile-first layout with flexible grid system
- **Accessibility:** Semantic HTML5, ARIA labels, skip links, and keyboard navigation
- **Modern Styling:** CSS variables for easy theming and maintainability
- **Contact Form:** Ready to integrate with backend services (Formspree, Netlify Forms, etc.)
